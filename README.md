# FinOps SaaS MVP

A simple SaaS platform for tracking cloud costs and financial operations.

## Features (MVP)
- **Project Tracking**: Manage multiple projects with defined budgets.
- **Expense Logging**: Track cloud spend across categories (Compute, Storage, etc.).
- **Dashboard**: Visualize total spend vs. budget.

## Technology Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS (CSS Modules)
- **Linting**: ESLint

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
